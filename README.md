# Mind Time

### Prerequisites & Installing

First, clone this repo:

```
git clone <link to this repo>
```

The backend of this single page application runs on a Rails API.

```
cd mhm-backend
```

Then run:
```
bundle install
```

Run the API with:
```
rails db:migrate

rails db:seed

rails s

`cd mhm-frontend`

Type `open index.html`

`cd ..`
